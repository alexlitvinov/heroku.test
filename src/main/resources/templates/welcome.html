Wellcome image builder

Usage

  • create template in /template folder
  • ask service

Service url - /service/getbase64

Service return - application/json

Service parameters

  • name - template name
  • data - json strong will be passed to template
  • height - height (int)
  • width - width (int)
  • quantity - must be (hi, mid, low)

Avaliable templates

  • smallSingleTemplate
  • midSingleTemplate
